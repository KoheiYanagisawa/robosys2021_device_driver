# robosys2021_device_driver
 - RaspberryPi4を使用しステッピングモーターを制御するデバイスドライバを作成しました。
 ## 動作環境
 - Ubuntu 20.04.3 LTS
 - RaspberryPi4
 ## 使用したもの
 - RaspberryPi4
 - 電源用USBケーブル　TypeC
 - ジャンパ線
 - ステッピングモーター(Quimat Nema 17)
 - ステッピングモーター制御IC(A4988)
